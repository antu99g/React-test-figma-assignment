# React Test

## Tech Stack

`React.js` `Material UI` `CSS`

## Run Locally

To run this project locally follow these steps :

1. Extract the .zip folder

2. Go to the project directory

```bash
  cd react-test
```

3. Install dependencies

```bash
  npm install
```

4. Start the server

```bash
  npm run dev
```
